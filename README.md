## Exercícios Práticos de JavaScript
Este repositório reúne meus **exercícios práticos diários de JavaScript**, feitos ao longo do meu estudo da linguagem.
A ideia aqui é **revisar e fixar os conceitos já aprendidos**, praticando todos os dias com exercícios variados.
Ao longo dos exercícios trabalho conteúdos como **estruturas condicionais, loops, arrays, métodos de array, strings, spread, destructuring, OOP e lógica de programação**.
Esse repositório também serve como um registro da minha **evolução e consistência no JavaScript** 🚀

## 📂 Estrutura do repositório
- `EX-01/` até `EX-101+/` – Exercícios variados cobrindo todos os temas estudados

Cada pasta contém seus arquivos `.js` e, quando necessário, arquivos `.html` para os exercícios com DOM.

## 🛠 Tecnologias
- JavaScript
- Execução via Node.js
- Navegador para exercícios com DOM

## ▶ Como executar os exercícios
### 🔹 Pelo terminal (Node.js)
1. Abra o terminal e vá até a pasta do exercício.
2. Execute o arquivo com:
```bash
node nomeDoArquivo.js
```
### 🔹 Pelo navegador (quando tiver DOM)
1. Abra o arquivo `.html` no navegador.
2. Se precisar, use o console (F12 → aba Console) para ver os resultados.
